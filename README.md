# Proyecto Palos Santos

## Objetivo

Desarrollar un software de gestión y una landing page premium conectada con la API de Instagram para sincronizar contenido de perfiles.

## Arquitectura Propuesta

- **Frontend (Landing Page)**: Web moderna, rápida y con estética premium (Vite + Vanilla CSS/React).
- **Dashboard de Gestión**: Panel para visualizar métricas, sincronizar posts y gestionar contenido.
- **Backend/Conector**: Integración con Instagram Graph API (Oficial) o servicios de scraping (Apify/RapidAPI).
